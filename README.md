# zig-toybox

Zig Toybox is a stdlib-only learning playground for Zig 0.16.

Build cursed little programs that make Zig's standard library, allocation model, errors, and build system easier to inspect.

## Start here

- Agents: read `AGENTS.md` before changing code or adding missions.
- Learners: read `LEARNERS.md` for ways to pair with an agent without hiding the machine.

## Project status

This repository is at the foundation stage. Missions, build wiring, and docs will be added incrementally.
